# Client (React + Vite)

Todo app frontend with login/register pages and a protected `/todos` route.

## Live Frontend
- CloudFront URL: https://d2vny3lui91j4l.cloudfront.net

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm test`

## Environment
Create `.env` from `.env.example`.

```bash
VITE_API_BASE_URL=http://localhost:3000
```
